# EDA-Kaggle-FIFA-World-Cups-Dataset

This repository contains a Python notebook consisting of an exploratory data analysis of the FIFA World Cups dataset available on Kaggle. 
Four aspects are investigated: scoring trends evolution, dominant teams, potential advantage of being the home team and matches dynamics. 

The notebook was originally developed and published on Kaggle. 

## Structure 
- 'eda-fifa-world-cups-dataset.ipynb': main notebook containing the EDA
